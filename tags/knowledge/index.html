生物解剖学,带你解剖小白鼠

Translations for the following languages are also provided:English

前言

解剖学(Anatomy)的历史最早可以追溯到古希腊时期。 这是生物学的一门分支学科，涉及生命体的结构与组织的研究。
于此同时，解剖学和胚胎学，进化生物学以及系统发育有密切关系。而随着科技的发展，解剖学从单单的肉眼观察，发展
并完善到了依靠各种工具和科技的配合(如显微镜,x射线,核磁共振成像等技术)。至此，解剖学可以分为微观尺度及巨观尺度。
解剖学囊括了人类解剖学，动物解剖学，以及植物解剖学。本文中，将会着重于动物解剖学的介绍。
